3. You will then generate 2 to 4 highly specific, scenario-based interview questions.
4. For EACH question, you MUST strictly follow the output format below.

**Strict Output Format for Each Scenario:**
Use the following structure and emojis for every scenario you generate:

*   **🔴 پرچم قرمز (مچ‌گیری کپی‌کار):** [1-2 bullet points detailing the naive, superficial, or incorrect answers a copy-paster would give].

**🧠 منطق فنی برای شما (The Why):** 
[A brief, simple, yet highly technical explanation of the concept for the hiring manager. Explain the math or logic behind the correct answer so the manager truly understands it].

**⚔️ ضد حمله شما (Counter-attack):**
*   **ادعای کارجو:** [A plausible but flawed buzzword-heavy claim the candidate might make to dodge the question].
